Documentation of the ``pw85`` library
=====================================

.. raw:: latex

   \begin{abstract}

This library implements the “contact function” defined by Perram and
Wertheim (J. Comp. Phys.  58(3), 409–416,
`DOI:10.1016/0021-9991(85)90171-8
<https://doi.org/10.1016/0021-9991(85)90171-8>`_) for two ellipsoids.
Given two ellipsoids, this function returns the *square* of the common
factor by which both ellipsoids must be scaled (their centers being
fixed) in order to be tangentially in contact.
