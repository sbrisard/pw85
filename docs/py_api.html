pypw85.spheroid(a, c, n, q=None)[source]¶

Return the quadratic form associated to a spheroid.

The spheroid is defined by its equatorial radius a, its polar radius c and the direction of its axis of revolution, n (array-like of length 3).

If q is not None, then it must be an array of length 6. It is modified in place.
